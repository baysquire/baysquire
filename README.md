### 🔬 Featured Deep Science Portfolio

#### 1. Physics-Informed Neural Networks (PINNs)
* **[Modeling Glomerular Solute Transport with 2D PINNs](https://github.com/baysquire/pinn-glomerular-filtration)**
  * Replaced traditional Finite Element Methods (FEM) with a deep learning architecture to model fluid dynamics and mass transport under hypertensive conditions.
  * Achieved <2.5% relative L2 error.
  * 📄 **Preprint:** [Read on Zenodo](https://doi.org/10.5281/zenodo.19921854)

#### 2. Quantum Computing & Cheminformatics
* **[Hybrid VQE for Molecular Ground State Optimization](https://github.com/baysquire/hybrid-vqe-drug-discovery)**
  * Engineered a Hybrid Variational Quantum Eigensolver (VQE) to compute ground-state energies of complex molecules, minimizing quantum circuit depths via classical gradient optimization.
  * 📄 **Preprint:** [Read on Zenodo](https://doi.org/10.5281/zenodo.20020464)

#### 3. Deep Learning for Drug Discovery
* **[Generative SMILES VAE for De Novo Molecular Design](https://github.com/baysquire/SMILES-VAE-Generative-Design)**
  * Developed a Character-Level Variational Autoencoder (VAE) pipeline with RDKit integration to automatically invent and screen chemically valid molecular structures.
* **[Tox21 PyTorch Geometric MPNN](https://github.com/baysquire/Tox21-GNN-Property-Prediction)**
  * Built a Message Passing Neural Network (MPNN) to predict chemical toxicity, specifically architected to handle the severe class imbalances and missing targets inherent in the Tox21 dataset.
